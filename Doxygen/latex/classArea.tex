{\footnotesize\ttfamily void Area\+::march\+In (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classUnit}{Unit}} $\ast$}]{unit,  }\item[{\mbox{\hyperlink{classArea}{Area}} $\ast$}]{from }\end{DoxyParamCaption})}



Called when a platoon marches into this area, determines whether or not a battle takes place and returns a \mbox{\hyperlink{classBattle}{Battle}} pointer if it does, otherwise returns N\+U\+LL. 

When a platoon marches into an area a few things could happen depending on the state of the area.

1) The area is unoccupied\+: The platoon claims the area for their country/alliance. A \mbox{\hyperlink{classTransport}{Transport}} route is set up between this area and the area the platoon came from. No battle occurs so N\+U\+LL is returned.

2) The area is occupied by allies\+: The platoon joins with the platoon already in the area. No battle occurs so N\+U\+LL is returned.

3) The area is occupied by enemies\+: The platoon that \char`\"{}marched in\char`\"{} is set as the attacker whilst the troop that preexisted is the defender. The attacking platoon fights the defending platoon in a battle. A pointer to the battle object is returned for resoltion.
