◆ marchIn()

void Area::marchIn ( Unit *  unit,
Area *  from 
)

Called when a platoon marches into this area, determines whether or not a battle takes place and returns a Battle pointer if it does, otherwise returns NULL.

When a platoon marches into an area a few things could happen depending on the state of the area.

1) The area is unoccupied: The platoon claims the area for their country/alliance. A Transport route is set up between this area and the area the platoon came from. No battle occurs so NULL is returned.

2) The area is occupied by allies: The platoon joins with the platoon already in the area. No battle occurs so NULL is returned.

3) The area is occupied by enemies: The platoon that "marched in" is set as the attacker whilst the troop that preexisted is the defender. The attacking platoon fights the defending platoon in a battle. A pointer to the battle object is returned for resoltion.

Parameters
platoonThe platoon marching into this area.
fromThe area that the unit came from
